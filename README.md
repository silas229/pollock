# Pollock

## Quick start

Copy `app.config.example.js` to `app.config.js` and edit the config.

Then run:

```sh
npm i -D
npm build:css
npm start
```

## Development

1. Start dev server

    ```sh
    npm run dev
    ```

2. Compile Tailwind Stylesheet

    ```sh
    npm run watch:css
    ````
